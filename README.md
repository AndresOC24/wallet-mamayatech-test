# Proyecto MamayaTech - Wallet
Desarrollar un MVP de billetera Web3 para MamayaTech que simplifique la adopción de blockchain en la empresa: tokenizará activos físicos como NFTs para un control transparente e inmutable (eliminando duplicidades en inventarios), gestionará tokens nativos [$Mamaya](https://sepolia.etherscan.io/token/0x1f380d35d2b84e3911aa5ebbdec9685f9163f58c) Token como sistema de recompensas internas (ganados por colaboración o metas y canjeables por beneficios), y los que se iran desarrollando mas adelante.
